# Личный Ассистент — Telegram Bot + Mini App

## Что умеет

- Расходы на жизнь (BYN) с разбивкой по дням / неделям / месяцам / годам
- Капитал (сколько денег лежит в разных местах, в $ и BYN)
- Профит по проектам (крипта, трафик и т.д.)
- Заметки
- Напоминания
- Быстрый ввод обычным текстом в чат
- Mini App для удобного просмотра

## Быстрый старт

### 1. Создай бота
1. Открой @BotFather в Telegram
2. Команда `/newbot`
3. Скопируй токен

### 2. Установка

```bash
cd personal_assistant_bot
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Настройка

Скопируй `.env.example` в `.env` и вставь токен:

```bash
cp .env.example .env
nano .env
```

### 4. Запуск бота

```bash
python bot.py
```

### 5. Mini App (пока демо)

Сейчас `miniapp/index.html` — это визуальная оболочка.
Чтобы она реально показывала данные, нужно:

1. Захостить папку `miniapp` (можно через GitHub Pages, Vercel, или просто ngrok + любой http-сервер)
2. Указать URL в `.env` → `MINI_APP_URL`
3. В @BotFather → Bot Settings → Menu Button → Configure menu button → указать тот же URL

Для полноценной синхронизации данных позже добавим небольшой FastAPI-бэкенд, который будет отдавать JSON по `initData` от Telegram.
